## Overview

This repository contains a curated collection of **Machine Learning projects** showcasing end-to-end workflows, including **data preprocessing, exploratory data analysis (EDA), model building, and evaluation**. Projects cover **regression, classification, and ensemble methods** with practical implementations using **Python, Scikit-learn, XGBoost, and CatBoost**. Emphasis is placed on **cross-validation, hyperparameter tuning, and best practices** to develop robust predictive models.

---

## Key Highlights

- **Data Preprocessing:** Handling missing values, feature encoding, normalization, and scaling.  
- **Exploratory Data Analysis (EDA):** Visualization, correlation analysis, and feature insights.  
- **Model Building:** Linear Regression, Ridge & Lasso, Logistic Regression, Decision Trees, Random Forest, Gradient Boosting, XGBoost, CatBoost, Naive Bayes.  
- **Model Evaluation:** Performance metrics, cross-validation, hyperparameter tuning, and best model selection.  
- **Tools & Libraries:** Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, XGBoost, CatBoost.
